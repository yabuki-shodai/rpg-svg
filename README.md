# RPG-SVG

RPG-SVG is a small web tool for generating RPG-style dialogue boxes as embeddable SVG images. You can enter your own lines of text, preview the result, and copy Markdown to place the generated image in a README or other documents. It is designed for creating playful profile intros, project headers, and retro game-style text presentations.

## Usage

1. Enter the lines you want to display in the RPG-style dialogue box.
2. Copy the generated Markdown snippet.
3. Paste it into your README, profile, or any other Markdown-supported document.

This can be used for GitHub profile READMEs, project introductions, or playful retro-style content blocks.
